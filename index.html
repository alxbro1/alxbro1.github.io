Proyecto 1: Priority Pass (NextJs API)

Este sitio web es una plataforma creada por una empresa de venta de cursos para promocionar su producto estrella. Alberga videos y audios de autoayuda e incluye un sistema de inicio de sesión al que se permite el acceso solo con autorización de administrador. El sitio está creado con Next.js y React, utiliza Zod para la validación de esquemas, PostgreSQL para la gestión de bases de datos y aprovecha JWT y cookies para la autenticación segura de usuarios.

Project 4: Game Vault (NextJs + TypeScript - NestJs + Typescript)

Este sitio web es una plataforma de comercio electrónico de juegos físicos y digitales, desarrollado como un proyecto en equipo utilizando la metodología Scrum. Como usuario, puedes interactuar a través de un cuadro de chat (que aparece una vez registrado e ingresado), comprar productos físicos y digitales, recibir y aplicar cupones de descuento, agregar artículos a tu carrito, realizar compras y ver todos tus pedidos con detalle. Como administrador, puedes realizar un CRUD completo sobre los productos, ver pedidos, administrar envíos, analizar datos de usuarios y chatear en vivo con los usuarios a través de WebSockets. Tecnologías utilizadas: Next.js, NestJS, React, TypeScript, PostgreSQL, Zod, ShadCN, Tailwind CSS, Drizzle ORM, NextAuth, JWT, bcrypt, Socket.io, Cloudinary, MercadoPago y más.
